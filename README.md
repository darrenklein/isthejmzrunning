# isthejmzrunning

What's up with the J/M/Z trains? This app will tell you right quick.

## Start the app

``` sh
export FLASK_APP=isthejmzrunning
export FLASK_DEBUG=true
flask run
```

Locally, this app runs on `localhost:5000`
